# Tap Savvy Lead Dashboard

## 1. Problem Statement

Many businesses generate large amounts of lead data, but analyzing lead quality, contactability, website adoption, and customer engagement manually becomes difficult and time-consuming.

The existing process lacked:
- Centralized lead monitoring
- Quick identification of high-quality leads
- Geographic analysis of business leads
- Website adoption tracking
- Efficient performance comparison

The raw dataset also contained:
- Missing values
- Duplicate records
- Inconsistent formatting
- Incorrect/null contact details
- Unstructured business information

Because of these issues, manual analysis became inefficient and decision-making was slower.

---

## 2. Objective of the Project

The main objective of this project is to build an interactive Power BI dashboard that helps users:

- Identify high-quality leads
- Analyze business performance
- Track contactability status
- Monitor website adoption rate
- Evaluate Google review performance
- Understand geographic distribution of leads

The dashboard aims to simplify lead analysis and improve business decision-making through data visualization.

---

## 3. Data Cleaning and Transformation

Before creating the dashboard, the dataset was cleaned and transformed using **Power Query**.

### Cleaning Steps Performed

- Removed duplicate records
- Handled missing/null values
- Standardized column formats
- Corrected inconsistent data entries
- Cleaned contact information
- Verified website availability data
- Organized categorical fields
- Renamed columns for better readability
- Filtered unnecessary records

These steps improved data accuracy and dashboard reliability.

---

## 4. Proposed Solution

To solve the above problems, a Power BI dashboard named **Tap Savvy Lead Dashboard** was developed.

The dashboard integrates lead-related data into a single interactive platform where users can:
- Filter data dynamically
- Compare lead quality
- Analyze trends visually
- Monitor KPIs in real time
- Explore location-wise lead distribution

This solution reduces manual analysis effort and improves reporting efficiency.

---

## 5. Technologies Used

- **Power BI** → Dashboard development and visualization
- **Excel / CSV Dataset** → Data source
- **Power Query** → Data cleaning and transformation
- **DAX** → KPI calculations and measures

---

## 6. Dashboard Features

### KPI Cards

#### Total Leads
Displays the total number of leads available in the dataset.

#### Avg Google Review
Shows the average Google review rating across businesses.

#### Website Adoption Rate
Indicates the percentage of businesses with a website.

#### Contactability
Measures the percentage of businesses that can be contacted.

#### High Quality Leads
Displays the count of leads classified as high quality.

---

## 7. Visualizations Used

### Lead Quality Distribution (Donut Chart)
Displays percentage distribution of lead categories.

### Lead Analysis (Bar Chart)
Compares lead counts across categories.

### Trend Analysis (Column Chart)
Shows comparison and performance trends.

### Business Information Table
Displays detailed business-level lead information.

### Geographic Distribution Map
Shows location-wise spread of leads.

---

## 8. Workflow of the Project

1. Collected lead dataset

3. Removed duplicates and handled missing values  
4. Created DAX measures and KPIs  
5. Designed dashboard visuals and filters  
6. Added interactive analysis features  
7. Published the final dashboard
